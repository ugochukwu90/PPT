## ALPHABETIC GRADING OF 100-SCORE-SCALE
#### PSUEDOCODE
```PY
START
INPUT score
IF score >= 90 THEN
OUTPUT 'Grade A'
ELSE IF score >= 80 THEN
OUTPUT 'Grade B'
ELSE IF score >= 70 THEN
OUTPUT 'Grade C'
ELSE IF score >= 50 THEN
OUTPUT 'Grade D'
ELSE IF score >= 40 THEN
OUTPUT 'Grade E'
ELSE IF score >= 0 AND >=39 THEN
OUTPUT 'Grade F'
```
